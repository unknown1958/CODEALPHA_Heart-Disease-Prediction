# Heart Disease Prediction using Machine Learning

## Description
A machine learning project to predict the presence of heart disease using patient medical data.

## Dataset
- UCI Heart Disease (Cleveland)

## Algorithms
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- XGBoost
